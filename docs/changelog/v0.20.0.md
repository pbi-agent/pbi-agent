# v0.20.0

**Release date:** 2026-06-21

## Changed

- Prompt enhancement now strips a model-emitted leading `<think>...</think>` trace before returning composer text while preserving later literal think tags ([6e4b716f](https://github.com/pbi-agent/pbi-agent/commit/6e4b716f7f2898f63708819954a72e074a5d9b46)).
- Plain `/release` now authorizes the full minor-release workflow by default, including local and remote unreleased commits plus push, merge, publish, and verification steps unless narrowed by the user ([d48d6aa3](https://github.com/pbi-agent/pbi-agent/commit/d48d6aa31fd9225255f6ff03be7b361464c95343)).
- Composer slash-command suggestions now open as an anchored upward overlay, request up to 200 commands, and scroll long command lists without covering the input row ([dbeda298](https://github.com/pbi-agent/pbi-agent/commit/dbeda298357aab0caa106a9ad8cb3312649ac8f5)).
- Browser-use skill documentation no longer carries a redundant `allowed-tools` section, so the skill catalog stays aligned with runtime tool policy ([bf9e3f0c](https://github.com/pbi-agent/pbi-agent/commit/bf9e3f0c3380c68638502e5ebb20804fe44483bd)).

## Fixed

- Explicit command and sub-agent references can now load disabled on-disk project sub-agents across prompt resolution, tool schema visibility, runtime invocation, and extension catalog paths ([3016352e](https://github.com/pbi-agent/pbi-agent/commit/3016352e820374efa2a12af8acaf866043e5ec5f)).

## Internal

- Updated task bookkeeping for the post-v0.19.0 release set ([6b746534](https://github.com/pbi-agent/pbi-agent/commit/6b746534c29ed8ebdc6f64985e02e98b44978077)).
