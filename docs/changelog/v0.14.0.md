# v0.14.0

**Release date:** 2026-06-17

## Added

- Added the `/code-quality-review` command and code-quality reviewer sub-agent for focused maintainability review workflows ([7c65cb4f](https://github.com/pbi-agent/pbi-agent/commit/7c65cb4f5715e50f8cd54e6600c613767cfd12a0)).
- Added Headroom-backed shell output compression so large command outputs stay compact while preserving shell status and truncation fields ([2cb4cd50](https://github.com/pbi-agent/pbi-agent/commit/2cb4cd50d7aa11a3b53bdad6f0bd3a7765434175)).

## Changed

- Updated `/fix-review` and `/code-quality-review` guidance with clearer review criteria and full-file scope for explicit filenames and paths ([2b189034](https://github.com/pbi-agent/pbi-agent/commit/2b1890347fb466161cfa13650c67bbf95799a853), [125ce6d3](https://github.com/pbi-agent/pbi-agent/commit/125ce6d31b7f1866a15b4be5176085a0bf8d7b9a), [6ae3e7ea](https://github.com/pbi-agent/pbi-agent/commit/6ae3e7ea6497777bb40041920bda47b22286430c)).
- Split the ChatGPT Codex WebSocket transport from backend orchestration for more maintainable subscription-provider behavior ([d217d64b](https://github.com/pbi-agent/pbi-agent/commit/d217d64bab71f686bfae6fb2e11fd5764eccf91d)).
- Renamed ChatGPT and GitHub Copilot subscription provider labels to remove parenthesized suffixes consistently across the CLI, docs, and web settings ([5e48d388](https://github.com/pbi-agent/pbi-agent/commit/5e48d38897849c7d93940e91d9e8d5136951bcfe)).
- Updated starter agent template memory guidance so generated agents rely on preloaded workspace memory instead of rereading it ([#315](https://github.com/pbi-agent/pbi-agent/pull/315)).

## Fixed

- Fixed GitHub Copilot Responses tool follow-ups by replaying local history instead of sending unsupported `previous_response_id` values ([8514fd1d](https://github.com/pbi-agent/pbi-agent/commit/8514fd1dcc44c3d49f34bd6825379774afc1fa9a)).
- Fixed maintenance cleanup for large SQLite purge batches that previously exceeded SQLite variable limits ([eb8ff17c](https://github.com/pbi-agent/pbi-agent/commit/eb8ff17c9f10f43a285df8f6b00b3a1a5ba16461)).
- Preserved OpenAI-compatible prompt cache usage details from generic chat-completions responses in token accounting ([9e4b950e](https://github.com/pbi-agent/pbi-agent/commit/9e4b950ea11bbe2357382b67d593d677f8dc1ec5)).
- Updated Starlette to 1.0.1 to include its malformed Host header URL handling fix ([#313](https://github.com/pbi-agent/pbi-agent/pull/313)).

## Internal

- Updated aiohttp to 3.14.0 ([#312](https://github.com/pbi-agent/pbi-agent/pull/312)).
- Recorded workspace memory and task bookkeeping updates from the release cycle ([10b77c79](https://github.com/pbi-agent/pbi-agent/commit/10b77c7988ce44b5503edeecf66543739c811ba0)).
