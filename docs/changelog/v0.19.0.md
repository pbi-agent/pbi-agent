# v0.19.0

**Release date:** 2026-06-21

## Added

- Added bundled pricing entries for `glm-5.2` and `minimax-m3`, with case-insensitive model catalog matching for exact and prefix lookups ([db03d601](https://github.com/pbi-agent/pbi-agent/commit/db03d601afb6fe8ff7dfb4bfa5b938e72193c224)).

## Changed

- Moved workspace memory to the final system prompt section so main and sub-agent prompts preserve prompt-cache-friendly prefixes while still including current memory ([377ac755](https://github.com/pbi-agent/pbi-agent/commit/377ac7553f9d4398cee4d11656bf266e16c42ef9)).

## Fixed

- Omitted empty `tools` and `tool_choice` fields from OpenAI-compatible Chat Completions requests when no tools are available, fixing no-tool provider calls such as prompt enhancement on GLM-compatible endpoints ([79a5a6bb](https://github.com/pbi-agent/pbi-agent/commit/79a5a6bbea9c09573f209d0310c1a37c9c211854)).
- Parsed OpenAI-compatible Chat Completions reasoning from GLM/DeepSeek-style fields and inline `<think>...</think>` content while keeping visible assistant text clean ([19daae28](https://github.com/pbi-agent/pbi-agent/commit/19daae2814d2e9b290697e4880be6e96ae661ddc)).
- Preserved raw tagged reasoning in Chat Completions replay messages so tool follow-up requests keep prior reasoning context while the UI still displays separated reasoning and response text ([7fbfd2b](https://github.com/pbi-agent/pbi-agent/commit/7fbfd2b120dab4f98f508f4eeedc8920c591bb9d)).
