# v0.21.0

**Release date:** 2026-06-22

## Added

- Added Codex-style command hooks across session, tool, compaction, stop, and sub-agent lifecycle events, with trust review through the CLI, web API, and Settings UI ([314d74fc](https://github.com/pbi-agent/pbi-agent/commit/314d74fc5280ad2b699765369331688b5d89ecfb), [2cd4fcdc](https://github.com/pbi-agent/pbi-agent/commit/2cd4fcdc06921bfbeac2c1d9b7c081e1954d637b)).
- Added Firecrawl-backed `web_search` and made `read_web_url` use Firecrawl scrape before falling back to markdown.new ([6f056117](https://github.com/pbi-agent/pbi-agent/commit/6f056117e5095930fa0b41762b671f1f0b61f7a9)).
- Added an `orchestrator` project sub-agent and updated `/orchestrate` so user-provided plans skip the planner and start directly at the worker ([31de49ee](https://github.com/pbi-agent/pbi-agent/commit/31de49ee10640dce1e35a2001cd3977f2a2c98e2)).
- Added desktop and sound notifications when an interactive session finishes while the user is AFK ([f79caa9b](https://github.com/pbi-agent/pbi-agent/commit/f79caa9b0a31fd6246f8abd3e2fb9d146aaa36ff)).

## Changed

- `/orchestrate` now enforces one sequential task flow with worker, reviewer/fixer, and code-quality/fixer gates, plus clearer sub-agent delegation rules ([3f87a0fd](https://github.com/pbi-agent/pbi-agent/commit/3f87a0fdb145e909995d788bc29a6b9bce8ea85d), [6c3f0a2c](https://github.com/pbi-agent/pbi-agent/commit/6c3f0a2c5941c2af68503cd8337ac31005ef4fc9), [3c1f9abb](https://github.com/pbi-agent/pbi-agent/commit/3c1f9abbd329e51bd87331a331bdf9e738683dc7)).
- Long user inputs in the session timeline collapse behind a safer expand control, reducing noisy transcript scrolling ([804cda77](https://github.com/pbi-agent/pbi-agent/commit/804cda77943b614523b49bc0377b83cad9bddb3d)).
- Command naming and documentation now consistently use `code-quality-review` for the code-quality review workflow ([49fff5bb](https://github.com/pbi-agent/pbi-agent/commit/49fff5bbff3338552adf0183d0effee1ed145699)).

## Fixed

- Corrected the sub-agent key used by `/orchestrate`, restoring the expected code-quality reviewer delegation ([1b0fec86](https://github.com/pbi-agent/pbi-agent/commit/1b0fec8651d682e9da564cf80da52420e29f356b)).
- Session-finished notifications now fire only on the AFK working-to-ready edge instead of treating successful interactive turns as ended live sessions ([c93cb3c7](https://github.com/pbi-agent/pbi-agent/commit/c93cb3c75148fd9b1a8129c4436ee98d9eed62c2)).

## Documentation

- Documented hooks configuration, trust review, managed hook semantics, dangerous bypass mode, matcher behavior, hook payloads, outputs, and current limitations ([2cd4fcdc](https://github.com/pbi-agent/pbi-agent/commit/2cd4fcdc06921bfbeac2c1d9b7c081e1954d637b)).
- Updated tool and customization docs for the Firecrawl-backed web tools and the current built-in `web` tool group ([6f056117](https://github.com/pbi-agent/pbi-agent/commit/6f056117e5095930fa0b41762b671f1f0b61f7a9)).

## Internal

- Refactored session/runtime, hook, provider, and tool execution internals to support hook lifecycle dispatch, guarded continuations, and clearer code structure ([70c296cb](https://github.com/pbi-agent/pbi-agent/commit/70c296cb46cc828e5fd8c100451fd07c24a4ebf4)).
